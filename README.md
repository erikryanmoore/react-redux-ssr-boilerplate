# React-Redux SSR Boilerplate

## Dependencies

-   [yarn](https://yarnpkg.com/en/docs/install#mac-stable)
-   [node 10.15.1](https://nodejs.org/en/download/)

## Installation

```bash
git clone git://github.com/erikryanmoore/react-redux-ssr-boilerplate.git
```

## Development

```bash
yarn dev-setup
# then
yarn dev
```

## Production

```bash
yarn prod
# then
yarn start
```

## Testing

```bash
yarn test
```
